# Juspay UI Dashboard

A React dashboard interface built for the Juspay UI Engineer assignment.

## Project Structure

- `public/`: Static assets, HTML entry point.
- `src/`: Source code - React components, pages, utilities, styles, and assets.
- `components/`: Reusable UI pieces (Header, Sidebar, Cards, etc.).
- `pages/`: Main pages (Dashboard, etc.).
- `utils/`: Mock data and helpers.
- `styles/`: Global styles and configuration.
- `assets/`: Logos, icons, images.
